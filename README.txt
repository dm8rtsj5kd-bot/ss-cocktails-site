S&S Cocktails 2025 — мини-сайт для барного кейтеринга

Как открыть локально (на компьютере или телефоне):
1. Откройте файл index.html в любом браузере (Chrome, Safari, Yandex, Edge).
2. Все ссылки и навигация будут работать офлайн.

Как выложить на GitHub Pages:
1. Создайте новый репозиторий на GitHub, например: ss-cocktails-site
2. Загрузите туда файлы из этой папки (index.html и README).
3. В настройках репозитория (Settings → Pages) выберите:
   - Branch: main
   - Folder: / (root)
4. Сохраните настройки — через минуту сайт будет доступен по адресу:
   https://ВАШ_НИК.github.io/ss-cocktails-site/

Файл index.html — это уже готовый сайт.
